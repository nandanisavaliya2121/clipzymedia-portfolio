# Clipzy Media Portfolio

A modern frontend portfolio website for Clipzy Media, a short-form content editing agency.

## Features

* Modern and responsive design
* YouTube Shorts and video showcase section
* Client results and portfolio highlights
* Creator growth workflow
* Contact section with direct links

## Technologies Used

* HTML5
* CSS3

## Purpose

This portfolio website was built to showcase Clipzy Media's content editing services, client work, and creator growth solutions.
